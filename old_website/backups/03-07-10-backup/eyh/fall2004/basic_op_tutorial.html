Logo Tutorial: Basic Operations

For each step, you should add the code as described and then click on the Draw button to see your code execute. Follow these steps to get started.

  1. Add a main procedure. This is required for Logo programs.
    2. procedure main
end
  2. Add code to draw a line. Try changing the value from 100 to other values, like 20, 333 and even -10. What happens when you let the turtle Draw?
    3. procedure main
    go 100
end
  3. Add code to turn your turtle left before drawing the line. The number 57 is the number of degrees to turn.
    4. procedure main
    left 57
    go 100
end
  4. Change the color of the line to red. Try adding the color line after the go line. Next, try it as shown here.
    5. procedure main
    color green
    left 57
    go 100
end
  5. Create your own custom procedure that you can call from the main procedure. Change the main procedure so that it calls the new procedure. 
    6. procedure main
    color green
    goto goLeft
end

procedure goLeft
    left 57
    go 100
end




  6. Add parameters (variables that can change) to your procedure and add the value to the main procedure.
    7. procedure main
    color green
    goto goLeft, degrees=57, steps=100
end

procedure goLeft
    left degrees
    go steps
end
  7. Add code to call your procedure again with different values.
    8. procedure main
    color green
    goto goLeft, degrees=0, steps=50
    color red
    goto goLeft, degrees=45, steps=50
end

procedure goLeft
    left degrees
    go steps
end
  8. Add code that repeats your procedure several times.
    9. procedure main
    color green
    goto goLeft, degrees=0, steps=50
    color red
    repeat 8, goLeft, degrees=45, steps=50
end
  9. Add a new procedure named goRight that does the same thing as goLeft, but for turning right.
    10. procedure goRight
    right degrees
    go steps
end
  10. Now, try the Load Maze button and write code so that the turtle draws a path that follows the maze that is shown.
